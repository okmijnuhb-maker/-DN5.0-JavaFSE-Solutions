**## Module 2 - Data Structures and Algorithms**



**### Exercise 2: E-commerce Search Function**

**- Implemented Linear Search and Binary Search**

**- Output:**

&#x20; **=== Linear Search ===**

&#x20; **Product found at index 4: Product ID: 104 | Name: Charger | Category: Accessories**

&#x20; **=== Binary Search ===**

&#x20; **Product found at index 3: Product ID: 104 | Name: Charger | Category: Accessories**

&#x20; **=== Time Complexity Analysis ===**

&#x20; **Linear Search: O(n) - checks every element**

&#x20; **Binary Search: O(log n) - divides array in half**

&#x20; **Binary Search is faster for large datasets!**
